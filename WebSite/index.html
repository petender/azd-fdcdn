Sample website for Azure FrontDoor and CDN demo

Loading Images from different Azure Services

This basic demo website allows you to get an idea how the performance of loading images is dependent on the actual Azure Service used. The 3 scenarios are Azure App Services, Azure Storage Account and Azure FrontDoor/CDN. While the first time you load the images, the difference won't be that big, it starts becoming a difference for the Azure CDN option after refreshing the website (Ctrl-F5). It should go from about 1000-3000 ms down to < 100 ms on average. Try to include a few learners, maybe even from different locations in the world, to connect to the website and experiment with the load times.
